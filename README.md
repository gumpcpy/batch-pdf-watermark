# 無限水印 Batch PDF Watermark

## 目的

同一個PDF檔案需要分發給多人的時候，

只要準備一個文字檔案，

就可以批量的生成加上水印的PDF

EX.

一份文件開會要發給參與會議的人員

列一份名單存成.txt (文字檔)

使用本程式可以批次生成多個加了水印的PDF

支援 繁體中文 簡體中文 英文


## 版本
